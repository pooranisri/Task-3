JavaScript Sorting Algorithms

Bubble Sort

Bubble Sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order.

Merge Sort

Merge Sort is an efficient, stable, divide and conquer sorting algorithm. It divides the input array into two halves, calls itself for the two halves, and then merges the two sorted halves.
